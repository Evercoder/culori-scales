# culori-scales

Color scales for [Culori](https://github.com/danburzo/culori).

## What's included

culori-scales packages:

* Cynthia A. Brewer's [ColorBrewer](http://colorbrewer2.org/) schemes;
* the [new default matplotlib colormaps](https://bids.github.io/colormap/) designed by Nathaniel J. Smith, St√©fan van der Walt, and Eric Firing;
